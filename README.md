Overview
This project includes 5 shell scripts to show off your Linux skills and your understanding of opensource software and how it works.
Scripts
1. System Identity Report (script1.sh)
This script will show information such as operating system, kernel, username, and uptime.
2. FOSS Package Inspector (script2.sh)
This script will check whether a package is installed or not and show information about it.
3. Disk & Permission Auditor (script3.sh)
This script will show directory permissions and disk usage.
4. Log File Analyzer (script4.sh)
This script will read log files, count the number of keywords, and show recent log entries.
5. Open Source Manifesto (script5.sh)
This script will generate a simple open-source message using user input.
